# DES Implementation

This directory contains a C++ implementation of the Data Encryption Standard (DES) completed as part of a summer school on symmetric-key cryptography. The implementation was developed using the provided project structure and helper functions, with the required DES algorithmic components completed as part of the coursework.

## Features

* DES encryption and decryption
* Initial and Final Permutations (IP/FP)
* Key schedule generation
* Feistel network implementation
* S-Box substitution
* Expansion and permutation operations
* Test case validation


## Learning Outcomes

This implementation provided practical experience with:

* Feistel network architecture
* Bit-level operations and permutations
* Round key generation
* DES encryption and decryption workflow
* Algorithm implementation and validation
