# AES Implementation

This directory contains a C++ implementation of the Advanced Encryption Standard (AES) completed as part of a summer school on symmetric-key cryptography. The implementation was developed using a provided code framework and helper functions, with the required algorithmic components completed as part of the coursework.

## Features

* AES encryption
* AES decryption
* Key expansion (Round Key Generation)
* SubBytes transformation
* ShiftRows transformation
* MixColumns transformation
* AddRoundKey operation
* Test case validation

## Learning Outcomes

* Understood the AES encryption and decryption workflow
* Implemented core components of the AES algorithm
* Gained experience with substitution-permutation network (SPN) architecture
* Worked with finite field arithmetic and round transformations
* Validated correctness using predefined test cases
